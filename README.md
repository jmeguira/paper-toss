# Paper Toss

Mobile-first projectile throwing game. Swipe to throw at a distant target, with wind and spin affecting flight. Built with Phaser 3 + TypeScript + Vite.

## Setup

```sh
npm install
sh setup.sh   # installs git hooks (pre-commit type check)
```

## Development

```sh
npm run dev
```

Opens a local dev server with hot reload. Use Chrome DevTools device emulation for mobile viewport testing.

## Build

```sh
npm run build
```

Outputs optimized production files to `dist/`.
